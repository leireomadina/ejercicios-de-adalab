Lenguajes de programación en Adalab

La formación técnica de Adalab se divide en tres módulos. En el primero aprenderemos a maquetar páginas web desde cero utilizando HTML5, CSS3, SASS y Bootstrap.

En el segundo módulo, nos adentraremos en el mundo de la programación propiamente dicha a través del lenguaje JavaScript (ES6), y aprenderemos a utilizar APIs de terceros.

Finalmente, en el tercer y último módulo aprenderemos a utilizar el framework de JavaScript llamado React, gracias al cual podremos crear SPAs sencillas.

Los lenguajes que estudiaremos son los siguientes:

  1. HTML y CSS
  2. SASS y Bootstrap
  3. JavaScript
  4. React

No sólo la formación técnica es importante. Cada día son más valoradas entre las empresas las llamadas Soft Skills: comunicación, trabajo en equipo, empatía, etc.

Además, utilizaremos una serie de herramientas para desenvolvernos en un ambiente técnico profesional de manera eficiente y productiva: Slack, Trello, GitHub o Gulp, entre otras.
